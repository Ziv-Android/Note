使用方法：
1. 启动oqc_app.exe, 选择 订单客户 和 产品型号 后点击 配置完成 按钮，进入下一页
2. 搜索设备按钮上方本地IP地址处是否为空，空白时需手动填写电脑本地IP地址在搜索设备按钮上方
3. 双目会显示 双目IP修改 按钮，点击后，等待按钮左侧提示修改成功
4. 点击搜索设备按钮，搜索出当前设备IP，单目为1个，双目为2个
5. 点击搜索出的IP地址列表，IP地址会自动填写在下方设备后
6. 用户名/密码默认为：admin/admin，需要修改时需要手动修改输入，无需修改时可直接跳过该步骤
7. 上述配置完成后点击自动测试或键盘按下F8后开始执行右侧测试列表中的测试项，列表内实时显示当前测试状态
8. 右侧测试项完成后，会自动切换至页面进行接口测试
9. 双目会从192.168.1.100开始，完成后会自动切换至192.168.1.101测试另一个
10. 全部测试项完成跳转至结果页，显示测试报告，并输出word文件在doc目录下
11. 测试项目失败时，测试过程终止，不进行后续测试
